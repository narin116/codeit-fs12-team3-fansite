## 📌 Project Overview

본 프로젝트는 국내 인디 락 밴드 **실리카겔(Silica Gel)** 을 주제로 제작한 팬 페이지입니다.

**코드잇 스프린트 FS 12기** 교육 과정 내 진행된
**[미니 프로젝트 - 밴드/아티스트 팬 페이지 만들기]** 과제입니다.

HTML과 CSS를 활용하여 정적 웹사이트를 구현하였으며,
Git/GitHub 기반 협업을 통해 실무와 유사한 개발 프로세스를 경험했습니다.

## 📌 협업 방식
### Git Flow에 따라 main ← dev ← feature 구조로 진행
1. 각자 담당 페이지를 feature 브랜치에서 작업
2. 작업 완료 후 Pull Request 생성
3. 팀원 코드 리뷰 진행
4. 전체 페이지 최종 승인 후 main 브랜치에 PR 후 merge
